Module OCamlMake_Common_Property.Container.LinkedHashSet

module LinkedHashSet: Container_LinkedHashSet
type 'a t
Type of linked hash sets.
val create : unit -> 'a t
Creates a new linked hash set
val add : 'a t -> 'a -> unit
Adds a value to the linked hash set.

Fails if the set already contains the given value.

val mem : 'a t -> 'a -> bool
Returns whether the linked hash set contains the given value.
val of_iterable : 'a Utils_Iterable.t -> 'a t
Creates a linked hash set containing all the elements in the iterable, in the same order.

If the iterable contains duplicate elements, they are de-duplicated.

val to_iterable : 'a t -> 'a Utils_Iterable.t
Returns an iterable that lists all the elements in the linked hash set, in insertion order.
val of_list : 'a list -> 'a t
Creates a linked hash set containing all the elements in the list.
val of_array : 'a array -> 'a t
Creates a linked hash set containing all the elements in the array.
