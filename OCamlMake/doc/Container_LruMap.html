Module Container_LruMap

module Container_LruMap: sig .. end
Maps that evict least-recently used (LRU) entries to ensure bounded size.
type ('k, 'v) t
Type of LRU maps.
val create : ?finalize:('k -> 'v -> unit) -> int -> ('k, 'v) t
Creates a new LRU map.
finalize : A method called when an element is evicted.
max_size : The maximum size of the LRU map, above which the least-recently used element is evicted.
val clear : ('k, 'v) t -> unit
Removes all the elements.
val get : ('k, 'v) t -> 'k -> 'v
Returns the value stored under the given key.
val remove : ('k, 'v) t -> 'k -> unit
Removes the value stored under the given key.
val put : ('k, 'v) t -> 'k -> 'v -> unit
Adds a key-value pair. Evicts the least-recently used element if necessary.
val to_iterable : ('k, 'v) t -> ('k * 'v) Utils_Iterable.t
Returns an iterable that lists all the elements in the map, in LRU order.
