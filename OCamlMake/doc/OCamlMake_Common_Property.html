Module OCamlMake_Common_Property

module OCamlMake_Common_Property: sig .. end
Implementation for file properties.

Properties are attached to Utils_File.t. Each value OCamlMake_Common_Property.t contains a list of generators that are used to generate properties. Properties are used to implement flags.
